# SystemJS Example

An example showing how to use SystemJS v0.21.5 to load modules that are written
in different formats.

```
├── arith.js
├── index.html
├── main.js
├── package.json
├── readme.md
├── system.0.21.5.js
└── values.js
```

- run `npm i` first here to install the dependencies.
- Then run a server in this folder and open `index.html`
